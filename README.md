# **_DevOps Sandbox_: Esteira de Automação e Containerização**

Este projeto prático simula o fluxo de trabalho de engenharia _DevOps_ para garantir a estabilidade e automação de uma _API_ em _Python_.

**Tecnologias Comprovadas neste Repositório:**
Containerização (_Docker_): Criação de um _Dockerfile_ otimizado utilizando imagens slim para reduzir a superfície de ataque e o tamanho do pacote.

**Infraestrutura como Código & _YAML_:** Escrita de _manifests_ declarativos para padronização de ambientes.

**_CI/CD (GitHub Actions)_**: Automação de pipeline que dispara a validação de sintaxe e testes de compilação a cada commit, impedindo que códigos quebrados cheguem à produção.

**Orquestração (_Kubernetes_)**: Arquitetura de um manifest de _Deployment_ focado em alta disponibilidade com suporte a múltiplas réplicas.
